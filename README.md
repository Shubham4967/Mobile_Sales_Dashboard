**Project Overview**
Developed an interactive Mobile Sales Analysis Dashboard in Power BI to monitor and analyze sales performance across different cities, brands, mobile models, payment methods, and time periods. The dashboard provides comprehensive insights into key business metrics and enables users to perform dynamic analysis through filters and slicers.

**Key Features**
Interactive dashboard with slicers for Mobile Model, Brand, Payment Method, Year, and Month.
KPI cards displaying:
Total Sales,
Total Quantity Sold,
Number of Transactions,
Average Price.
City-wise sales analysis using map visualization.
Monthly quantity trend analysis.
Payment method distribution analysis.
Brand-wise and model-wise sales comparison.
Day-wise sales performance analysis.
Month-to-Date (MTD) sales tracking.
Same Period Last Year (SPLY) analysis with Year, Quarter, and Month comparisons.

**Tools & Technologies**

**Key Insights**
Identified top-performing mobile brands and models.
Analyzed sales distribution across various cities.
Compared current sales with previous year's performance.
Monitored Month-to-Date sales trends for better business tracking.
Evaluated customer payment preferences and transaction patterns.
